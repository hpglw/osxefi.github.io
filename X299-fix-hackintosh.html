【经验】聊聊X299安装黑苹果中所遇到的问题与解决

发表于 2019-11-15 更新于 2022-07-01 分类于 教程

以一台i7-7800x配微星X299 SLI PLUS+64GB 海盗船DDR 4 3000Mhz为例。

1，卡加号


End RandomSeed
++++++++++++++++++++++++++++

解决尝试：
1，config.plist中boot args加入slide=129
2，在CLOVER / drivers64UEFI 中加入OsxAptioFix2Drv-free2000.efi文件
3，BIOS中降低内存频率，尝试2400Mhz。
4，关闭CFG Lock，bios中说明内容为”关闭或开启MSR 0xe2“

2，卡在[ PCI configuration begin ]不动

解决尝试：
1，config.plist中boot args加入npci=0x2000

3，卡在bash: /etc/rc.server: No such file or directory

解决尝试：
1，在CLOVER / drivers64UEFI 中删除HFSPlus.efi，加入VBoxHfs-64.efi

其它建议：

1，config.plist APCI项中Patches全部留空
2，KernelAndKextPatches中KernelToPatch加入

3，SMBIOS机型为iMacPro1,1
4，KernelPm开启，为true。
