RAZER 雷蛇 Blade Stealth 灵刃潜行版 12.5 4K触屏笔记本安装黑苹果Catalina 10.15.7 Hackintosh

发表于 2020-12-30 分类于 笔记本 ， Razer Blade_雷蛇
配置

Blade Stealth 灵刃潜行版 12.5
电脑型号 Razer Blade Stealth 笔记本电脑
操作系统 Windows 10 64位（Version 1909 / DirectX 12）
处理器 英特尔 Core i7-7500U @ 2.70GHz 双核
主板 Razer（7th Generation Intel Processor Family I/O - 9D58 笔记本芯片组）
显卡 英特尔 HD Graphics 620 ( 128 MB / Razer USA )
内存 16 GB ( 三星 LPDDR3 1867MHz )
主硬盘 三星 MZVLV512HCJH-00000 ( 512 GB / 固态硬盘 )
显示器 夏普 SHP144D ( 12.5 英寸 )
声卡 瑞昱 @ 英特尔 High Definition Audio 控制器
网卡 DW1560

该笔记本是轻薄本，12.5寸显示屏4K分辨率，7代HD620核显，无独显，己更换DW1560博通黑苹果免驱无线网卡，正常使用WIFI蓝牙，支持隔空和接力。两个USB3接口，一个雷电3接口，己定制，但雷电3仍不支持即插即用，需要插着重启后才能正常使用，另外不支持HUB的HDMI输出，支持本机的HDMI的输出，完美4K 60Hz。

演示视频：https://www.ixigua.com/6911995883156406798/

安装前注意：

  • 利用本引导的cfglock.efi（OC引导按空格显示）解锁CFG LOCK原生CPU电源管理，如果不解锁，请开启OC里config.plist > Kernel > Quirks里的AppleCpuPmCfglock和AppleXcpmCfglock，否则将无法进入macos。详见【老吴黑苹果】解锁CFG LOCK的新姿势
