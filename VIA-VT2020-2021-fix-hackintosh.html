黑苹果VIA VT2020 / 2021(0x0441)声卡驱动方法

发表于 2019-11-04 更新于 2019-11-15 分类于 教程

以台式机技嘉Z77X-UD3H主板为例，内置VIA VT2020 / 2021(0x0441)声卡芯片，采用APPLEALC驱动。


查询Supported codecs · acidanthera/AppleALC Wiki · GitHub

在支持的列表中，id有三个 5 7 9。

驱动方法：
下载：applealc.kext
https://github.com/acidanthera/AppleALC/releases/download/1.4.3/AppleALC-1.4.3-RELEASE.zip

将applealc.kext放在CLOVER / KEXTS / OTHER里面

用Clover Configurator打开config.plist
几个地方：


注入ID为9

如遇其它主板ID 9不行，就换 5或者7。

以上就是驱动方法，关于voodoohda，目前测试不通过，没有声音，也建议使用applealc驱动。
