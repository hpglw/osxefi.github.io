发表于 2022-02-14 更新于 2022-08-10 分类于 笔记本

配置

电脑型号 蓝天 P770DM 笔记本电脑
处理器 i9-9900t es @ 1.70GHz 八核
主板 蓝天(CLEVO) P7xxDM(-G)（100 Series/C230 Series 芯片组 Family - A145）
显卡 NVIDIA Quadro K3100M ( 4 GB / 蓝天(CLEVO) )
内存 16 GB ( 三星 DDR4 3200MHz )
主硬盘 三星 MZVLB512HBJQ-000L2 ( 512 GB / 固态硬盘 )
显示器 友达 AUO219D ( 17.2 英寸 )
声卡 英特尔 High Definition Audio 控制器
网卡 鈺硕 Killer e2400 Gigabit Ethernet Controller / 蓝天(CLEVO)

说明

这是一台比较魔幻的笔记本，也不能说是笔记本，只能说是带个电池的台式机，有多魔幻，cpu是es版的，还是个9代cpu，然后主板是200系的，也就是魔改bios的。然后没有核显，独显竟然是开普勒核心的NVIDIA Quadro K3100M，神奇的是竟然可以驱动最高安装macos big sur，还要加个三星的pm981在里面，需要主动禁用，用另外的硬盘来安装，这三大要素在里面，竟然安装成功了，完全是黑苹果机型中的一个奇葩。
