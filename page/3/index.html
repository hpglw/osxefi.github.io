主板 华硕 WX:e20gs7（100 Series/C230 Series 芯片组 Family - A148）
显卡 英特尔 UHD Graphics 630 ( 128 MB / 华硕 )
内存 16 GB ( 酷兽 DDR4 2666MHz )
主硬盘 HS-SSD-E100N ( 512 GB / 固态硬盘 )
显示器 飞利浦 PHL08FA PHL BDM4350 ( 43.2 英寸 )
声卡 瑞昱 ALC887 @ 英特尔 High Definition Audio 控制器
网卡 Atheros AR9485 Wireless Network Adapter / 戴尔

说明

HDMI显示器4k的，所以要在bios中把显存改为128mb，其实这个引导并没有什么适配性，因为主机也不知道是什么型号，只知道是华硕的，华硕默认是关了cfg lock的，只是主板是魔改了的，如果对于其它配置，cpu同样为qqlt也就是i9-9980hk的，也许能适用，一般为100系或者200系主板魔改用。
Cpu变频是完全正常的最低700mhz到最高睿频。
其它基本驱动的。
魔改的主板加cpu有很多不确定性，安装黑苹果不是太容易，谨慎考虑。

华硕Vivo AIO 22 V222UA i5-8250U一体机黑苹果引导OC 7.7 MONTEREY 12.1

发表于 2022-02-10 分类于 台式机 ， 一体机

配置

华硕Vivo AIO 22 V222UA一体机
处理器 i5-8250U
主板 华硕Vivo AIO 22 V222UA
内存 4 GB
主硬盘 金士顿256gb
主显卡 uhd 620
显示器 1080p内置
声卡 alc256
网卡 rtl8168有线网卡+杀手无线网卡

说明


如果关闭cfg lock，在oc引导页按空格，选modGRUBShell，输入
setup_var_3 0x563 0x0
之后回车，然后输入reboot重启。这一步一定要做，因为引导是关cfg lock配置的。

声音如果没有，要在macos关机，开机后直接进macos，麦克风可以用，耳机不一定，自带扬声器是可以响的。

摄像头是有的

自带无线网卡是杀手无线网卡wifi不能用，但蓝牙是好的，蓝牙不一定可以连所有设备。

亮度不可调节

核显uhd620，声卡，有线网卡，蓝牙，变频等驱动。
