技嘉P65笔记本安装黑苹果系统Catalina 10.15.7

发表于 2020-11-22 分类于 笔记本 ， Gigabyte_技嘉

一台技嘉P65笔记本，配置为i7-7700HQ,16GB内存，512G固态,核显HD 630，独显GTX 1060（需要屏蔽），1080P分辨率，INTEL无线网卡加REALTEK有线网卡。

看配置是Kaby Lake架构CPU，核显是HD630，相对来说比较容易安装，主要是分辨1080P，这个只要HD630的config就基本没问题，系统选用CATALINA 10.15.7，目前来说，BIG SUR对黑苹果来说不建议，也没什么用处。

这次选用比较成熟的CLOVER引导，主要为了方便，其实OC差别不大。

首先BIOS设置：

1，确认SATA模式为AHCI

2，关闭安全启动

BIOS其它基本保持默认即可。

然后恢复老吴的10.15.7恢复版镜像，建立CLOVER引导

这里注意，CLOVER的版本要最新的5.1，内存修复不能使用AptioMemoryFix-64.efi，要用OcQuirks.efi和OpenRuntime.efi，这对于10.15系统来说AptioMemoryFix己经不行了，通常卡+++++++号都是因为这个引起的。

使用OcQuirks.efi和OpenRuntime.efi需要在config.plist勾选以下选项。

其它按HD630的核显config配置，需要注意的是，这个笔记本外放有问题，APPLEALC声卡驱动和VOODOOHDA万能驱动都只能支持耳机，外入无声卡，所有如果用APPLEALC，ID为15,用VOODOOHDA就默认就行了。这里我用了VOODOOHDA驱动，比较方便。

另外进了系统后，需要驱动WIFI的，请放入AirportItlwm.kext，具体用法请参考：https://openintelwireless.github.io/itlwm/Installation.html#itlwm

安装完成技嘉P65 i7-7700HQ 核显 声卡耳机无外放 电源电量 触控板基本 WIFI 蓝牙 CLOVER 10.15.7

EFI引导下载：http://pay.k61.org/share_1453.html

外星人Alienware 17 R5黑苹果安装记录 i7-8750H 独显驱动GTX 1060

发表于 2020-11-18 分类于 笔记本 ， Alienware_外星人

安装外星人alienware 17 r5笔记本黑苹果macos 10.13.6记录，只有独显GTX 1060.
先看配置：

CPU为i7-8750H,硬盘256G的东芝固态，16G内存，GTX 1060独显，UHD 630核显（系统己屏蔽）。


Killer E2500网卡

由于核显没有，N卡GTX 1060只能安装macos 10.13.6版本，使用的是老吴制作的MACOS 10.13.6恢复版镜像，有需要的可以联系我，提供下载地址。

macos 10.13.6版本比较旧，所以使用CLOVER引导即可，比较稳定，没有必要使用OPENCORE。

整体安装过程所遇到的问题只有一个，就是客户的硬盘，因为本身自带的256G己经安装了WIN，而且空间不足于安装黑苹果，所以客户加一个3.5的机械硬盘1T的，但这个硬盘比较难搞，当硬盘模式为AHCI的时候（苹果必须！）不能识别到硬盘（CLOVER引导），当RAID ON模式的时候才能识别，但显然是安装不上去的这个模式。

所以叫客户换了一个128G的东芝固态，然后从恢复镜像，到安装完成，不到15分钟。

该配置主要要留意的是，独显的记得打上WEB DIRIVER驱动，CLOVER的CONFIG里要启用独显驱动和WEB DRIVER，其它并没有太多要求。

OC: Failed to load configuration报错的解决方法

发表于 2020-07-15 分类于 教程

OPENCORE启动报错：OC: Failed to load configuration！

大致意思是OC配置文件被损坏了，有可能是config.plist，这时进入efi/oc，看到config.plist，打开是空白的，解决方法是重新恢复备份的config.plist，如果没有备份的，就重新做个config.list吧。

引起这个错误的大体上是双系统，WIN的杀毒软件，或者磁盘错误，都有可能导致config.plist损坏，建议备份。
