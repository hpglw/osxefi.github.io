配置

CPU (英特尔)Intel(R) Core(TM) i5-10400 CPU @ 2.90GHz(2904 MHz)
主板 华硕 ROG STRIX B460-I GAMING
内存 16.00 GB ( 2133 MHz)

i5-9400F - 微星 B360M MORTAR - RX 5500 XT 黑苹果安装EFI OC7.9 MONTEREY 12.4

发表于 2022-07-12 更新于 2022-09-08 分类于 台式机 ， INTEL

配置

处理器 英特尔 Core i5-9400F @ 2.90GHz 六核
主板 微星 B360M MORTAR (MS-7B23)（B360 芯片组）
显卡 AMD Radeon RX 5500 XT
内存 16 GB ( 三星 DDR4 2666MHz )
主硬盘 西数 WDS100T3X0C-00SJG0 ( SN750) (1 TB / 固态硬盘)
显示器 LG GSM5BD3 LG ULTRAGEAR ( 31.5 英寸 )
声卡 瑞昱 ALC892 @ 英特尔 High Definition Audio 控制器
网卡 英特尔 Ethernet Connection I219-V / 微星

说明

原本买的独显是撼讯RX 5500 XT，竟然翻车了！死活无法显示，进黑苹果就黑屏，然后一闪一闪的，能看到安装界面，dp和hdmi都一样，无数尝试后，换了一个华硕的rx 5500xt，完全正常了，建议黑苹果的，不要买这种不知名品牌的显卡。

主板msi的b360m mortar非常适合黑苹果了，有rx5500 cpu带不带核显都不是问题，所以9400f够了。
sn750真的太适合黑苹果了，速度一流，兼容性100%基本上，安装黑苹果墙裂推荐西数nvme固态。
这套配置基本上把性能全部发挥出来了。
