【老吴黑苹果】如何选择合适的版本，macOS系统各版本区别

发表于 2021-03-09 分类于 教程

macOS High Sierra 10.13.6


2016-2017年发布，Nvidia独显驱动最高支持到这个版本，一般为N卡配置的台式机安装版本，支持APFS文件系统，版本比较久，目前比较多的软件己经不再支持。
建议2代台式机，3代-4代笔记本，注意，如果是N卡独显，只能安装这个版本，注意，10代CPU不支持！！

macos Mojave 10.14.6


2018年发布，增加暗黑模式，主流苹果的FPCX最新版，XCODE己经不再支持，其它软件兼容性还可以，建议3代到5代笔记本安装此版本。

macos Catalina 10.15.7


2019年发布，首个64位macOS，不支持32位应用程序，目前基本上所有的主流软件都兼容，建议5代到10代笔记本安装，目前3代仍可以支持安装。

macOS Big Sur


2020年发布，最新支持ARM处理器的MACOS版本，是苹果取代INTEL处器器的开端，目前可能存在部分软件兼容性问题，但整体运行良好，建议6代以上，目前最高支持10代ICE LAKE处理器。不建议3代安装。
