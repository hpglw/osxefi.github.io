戴尔 OptiPlex 5040 i5-6500黑苹果引导EFI OC7.7 Monterey 12.1

发表于 2022-02-25 更新于 2022-02-26 分类于 台式机 ， INTEL

配置

电脑型号 戴尔 OptiPlex 5040 台式电脑
处理器 英特尔 Core i5-6500 @ 3.20GHz 四核
主板 戴尔 0T7D40（100 Series/C230 Series 芯片组 Family - A146）
显卡 英特尔 HD Graphics 530 ( 128 MB / 戴尔 )
内存 16 GB ( 海力士 DDR3L 1600MHz / 记忆科技 DDR3L 1600MHz )
主硬盘 台电 256GB NP900-2280 (256 GB / 固态硬盘)
显示器 HFC0000 D27 Art Lite ( 27 英寸 )
声卡 瑞昱 @ 英特尔 High Definition Audio 控制器
网卡 英特尔 Ethernet Connection I219-V / 戴尔

说明


安装视频：https://www.bilibili.com/video/BV1pS4y1k73A/

引导可支持Catalina Big Sur和monterey。此配置是4k显示器，需要修改dvmt,方法是在oc引导页，按空格，选modGRUBShell，进入命令行，输入以下指令：

改DVMT Pre-Allocated为128mb（以支持4k显示器）
setup_var_3 0x350 0x4

改DVMT Total Gfx Mem为max
setup_var_3 0x351 0x3

关闭CFG lock
setup_var_3 0xAF 0x0

以上完成后，重启再进macos，一定要做，否则会卡死进不去。
有intel的wifi网卡，所以无线是可以用的，蓝牙要搜比较久，但一般也能用。

调整DVMT Pre-Allocated为64MB:

调整DVMT Total Gfx Mem为MAX:
setup_var_3 0x796 0x3

完成后输入reboot重启

之后再进macOS，以上步骤一定要弄，否则会卡死。
