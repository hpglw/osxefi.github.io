再谈谈三星PM981和EVO 970 PLUS安装黑苹果的问题

发表于 2022-08-29 分类于 教程

虽然说这两款NVME固态的确性价比不错，很多笔记本，如小米，戴尔，联想都选用了这款硬盘。

更新：PM981不支持！不用折腾了！

PM981型号开头为MZVLB，只要有MZLVB就是PM981，还有PM981A两种。但就安装黑苹果来说，目前为止不可以，PM981和PM981A绝对是不行的。国内有些论坛说己经解决，那是不可信的，因为老吴己经安装太多了，无非表现都是开机能进系统，但会莫机卡死，自动重启等一系列奇怪的问题，反正就是不能正常使用，也用过各种说可以解决的方法，无果！从国外的大神反应来看，都是保持沉默，要么建议更换更换其它硬盘

如果你的笔记本中有PM981，也不是说一定安装不了，一般现在的笔记本都可以安装两个硬盘，只要将黑苹果安装在另一块上面，然后进黑苹果之后屏蔽PM981即可，屏蔽的方法很多，有DSDT法，也有开机不挂载法，老吴认为开机不挂载法是最方便的，也是最简单的
教程可以看：https://hpglw.com/diable-pm981-on-hackintosh.html

另外还有一个三星EVO 970 PLUS，目前这一块可以通过刷新最新的固件，安装黑苹果是没有问题，教程在这里：https://hpglw.com/970-EVO-Plus-firmware-flash.html
但是，目前发现了在macos catalina 10.15.2版本中，这块硬盘又出问题了！！再次不兼容！具体表现就是无法进系统，卡在苹果LOGO上，还没有解决方法，只能用10.15.2以下的macos版本。

老吴建议，三星这两款固态的，都不要安装黑苹果，就算安装，也只能在其它硬盘上装，然后还是屏蔽掉，也许只能等固件更新，毕竟三星也不会为了黑苹果而去修改固件，更多的还是为windows系统考虑。

黑苹果App Store应用商店账号无法登录的问题

发表于 2022-08-29 分类于 教程

无论是黑苹果还是白苹果，有时会出现，明明icloud可以登录，但app store无法登录。

明明在app store里登录了之后，两步验证也过了，然后还是显示未登录状态。

尝试换一个apple id登录，可能就没有问题，这是为什么呢？

老吴觉得有几个原因：

一是，没有本地有线网卡：
很多笔记本可能只有一个无线网卡，这时，我们需要在oc或者clover里加入这两个文件，仿冒一个有线网卡出来。
SSDT-RMNE.aml和NullEthernet.kext,在这里下载到：
链接: https://pan.baidu.com/s/1UmLZ4PdpfkL0kObOQ21PgA?pwd=8nk8 提取码: 8nk8
SSDT-RMNE.aml放在OC的ACPI里，NullEthernet.kext放在kexts文件夹里，然后oc的config.plist添加并启动这两个。


这个方法同样适用于黑苹果连内置无线网卡都无法驱动，只能外接usb无线网卡的用户，可以尝试。

二是，确实是网络问题
这个就比较玄乎了，如果可以，尝试电脑连手机热点，或者换一个网络环境，看能否登录上。

三是，这个是本人验证过的
同样也属于网络问题，如果你的apple id的地区是中国以外的，如很多人的是美区或者其它地区的apple id，icloud可以登录没问题，但app store是无法在中国的网络下登录的，具体你懂的。
如果你的apple id是其它地区的，你需要一把梯子，当你架好之后，就会发现，其实早登录上了，只是因为app store无法访问其它地区，所以不显示而己，具体怎么架梯子，自行脑补了。

除了这些，app store无法是黑苹果还是白苹果，无论有没有三码，或者黑苹果机型有没有正确的序列号，其实都不影响的，主要还是这几个原因，大多数是1和3的原因，具体自行检查一下。苹果不会有事没事去封一个id什么的，禁止你登录大多数是密码安全性考虑而己，这种情况很少。苹果也不会因为你是黑苹果系统而去封你的id什么的，放心使用。

机械键盘在黑苹果中失灵，按键延迟，迟滞，跳键，反应慢等打字问题的解决方法

发表于 2022-08-25 分类于 教程

很多机械键盘在黑苹果中都可能存在失灵，按键延迟，迟滞，跳键，反应慢等打字问题，具体表现为键位错乱，无故按键操作，打字反应慢等不正常的现象。而普通的键盘则没有这种问题。
这个其实和黑苹果的引导配置没有关联的，如果usb端口是好的话。

因为机械键盘和普通的薄膜键盘在固件和芯片上都有差别的，一般都有不同的键盘固件，而且是可更新的。
这时，我们就要尝试一下更新机械键盘的固件，这些操作都需要在windows下进行。
下载“ikbc键盘Mac系统触发程序”里面有很多不同类型的机械键盘固件，可以根据自己的机械键盘芯片进行更新。
链接: https://pan.baidu.com/s/1eEy4pyiPs2R3W6p3Qca9Qw?pwd=p63p 提取码: p63p
–来自百度网盘超级会员v6的分享

如果不更新固件，也可以尝试在windows下按fn+Scr 就可以全键无冲突，这个可能需要参考键盘的说明书设置。
