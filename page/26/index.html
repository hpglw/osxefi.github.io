解决黑苹果关于本机处理器识别错误的问题

发表于 2019-11-05 更新于 2019-11-15 分类于 教程

黑苹果安装完之后，关于本机的处理器多数显示未知，或者i7识别为i9这些的错误，网上的教程大多数是修改系统的语言文件，但这样做并不太好，除非严重强迫症。

其实有一种方法是比较简单的，就是修改clover的config.plist文件。

这里我们用到Clover Configurator
下载地址：https://mackie100projects.altervista.org/download-clover-configurator/

打开config.list

在CPU设置一栏的类型处，填入正确的CPU类型保存重启就可以了。

参考的CPU类型代码：

  • 0x0309 = Core i3
  • 0x0407、0x0601、0x0605 (优先使用) = Intel Core i5
  • 0x0701、0x0507、0x0705 (优先使用) = Intel Core i7
  • 0x1005= Intel Core i9
  • 0x0505 = Dual-Core Intel Xeon
  • 0x0510 = Quad-Core Intel Xeon
  • 0x05A5 = Quad-Core Intel Xeon
  • 0x0506 = 6-Core Intel Xeon
  • 0x0a01= 8-Core Intel Xeon E5

避免使用三星PM981安装黑苹果

发表于 2019-11-05 分类于 教程

目前EVO 970 PLUS己经有新的固件，可以安装黑苹果了。
http://k61.org/970-EVO-Plus-firmware-flash.html

目前就是这个PM981，一般型号为MZLVB开头的。这个型号的SSD的目前不能安装黑苹果，哪怕作为资料盘也不行，在AMD处理器的台式机上，会直接重启死机。

在INTEL处理器的机子上，会出现反应慢，卡死等不正常现象

所以要安装黑苹果的，把PM981取下来吧，在三星没有修复固件之前，没有任何解决方法。
