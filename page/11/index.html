主显卡 UHD630+GTX 1650
显示器 1080P 144HZ
声卡 ALC269
网卡 英特尔 Wireless-AC 9462

说明

注意一点，可能大部分这个型号的都有镁光s2200固态，所以引导里默认屏蔽了第一块nvme固态，如果你内置的不是镁光s2200，需要到EFI/OC/ACPI里删掉SSDT-DPM981.aml
这个笔记本可以关cfg lock，这个引导是需要关掉的，方法是在oc引导出来的时候，按空格，选cfg lock，然后按y，再按回车，重启一下，再进macos,不关的话会卡在开头不动的。
需要蓝牙就安装big sur
144hz屏，不要开hidpi，否则可能会花屏，开了也不能上144hz了
驱动了核显，声卡，麦克风，wifi，蓝牙，触控板，电池，usb，type c一个，变频完美，外接显示器不要想，直连独显的。

【老吴黑苹果】macOS Monterey 12.2.1(21D62)正式版恢复镜像发布

发表于 2022-03-08 更新于 2022-07-06 分类于 镜像

迟来的发布，macOS Monterey 12.2.1(21D62)正式版恢复镜像

大小：14.28GB

镜像格式：rdr(内附恢复工具和教程)

下载方式：百度云

介绍：macOS Monterey 12.2.1 更新可以通过系统偏好设置的软件更新部分，在所有符合条件的 Mac 上下载。

根据苹果的发布说明，macOS Monterey 12.2.1 解决了一个导致连接到 Mac 的蓝牙设备消耗过多电池电量的错误 Bug。

在今年 1 月底发布 macOS Monterey 12.2 更新后，许多用户进行投诉，他们在睡眠模式下经历了过度的电池消耗，而罪魁祸首似乎是那些经常唤醒他们所连接的 Mac 的蓝牙配件。

一些受影响的用户在安装 macOS 12.2 更新后，看到他们的 Mac 在睡眠模式下的电池续航从 100% 下降到 0%，基于英特尔的 Mac 和基于 Apple Silicon 的 Mac 都受到影响。当蓝牙被禁用或蓝牙配件与 Mac 断开时，该问题就会停止，这对用户来说不是一个理想的解决方案。

苹果昨天在 macOS Monterey 12.3 Beta 2 中解决了这个问题，但现在 Mac 用户将不需要等待 macOS Monterey 12.3 发布来解决他们的蓝牙相关电池耗尽问题。

macOS Monterey 12.2.1 还修复了一个 WebKit 漏洞，该漏洞可能允许恶意制作的网页内容导致任意代码执行。苹果表示，它知道有报告说这个问题可能被积极利用，所以 Mac 用户应该尽快更新到新软件。

最后，苹果还为 macOS Big Sur 和 Catalina 上的用户发布了新版本的 Safari 15.3，估计是为了解决 macOS Monterey 12.2.1 中修复的同一个 WebKit 漏洞。

下载地址：http://wpa.qq.com/msgrd?v=3&uin=349926524&site=qq&menu=yes
