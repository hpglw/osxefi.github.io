记录黑苹果安装中遇到的错误与解决2020-12-02

发表于 2020-12-03 分类于 教程
4K显示器不能自动开启hidpi的问题

一般来说只要是4K的显示器，hidpi不用手动开启，自动的，但这也跟连接线有很大的关系，首先线材的质量要过关，其次要支持4K以上的，如HDMI支持8K的版本，有些DP线不合格也是不会自动开启的，所以遇到这种问题，换根线试一下

This version of Mac OS X is not supported on this platform!

字义上就很好理解了，这个平台的配置不适合安装这个版本的macos，主要是机型SMBIOS没有选好。
例如2019年的imac机型是无法安装macos 10.13.6的，反之2012年的也无法安装最新的macos版本，换一个机型就可以了，这个可以在config里设置。

Realtek 8125网卡的问题

除了要放最新的驱动，网卡设置里不能自动获取DCHP，需要手动设置IP，才能上网。

4790K核显加速引起独显崩溃播放视频变成绿屏

换个机型就好了

多数服务器主板黑苹果关机变重启的问题

例如，X79 X99 X299这类服务器主板，安装黑苹果可能会出现关机变重启的问题，对于这个问题，实在没有太好的解决方法，主要是此类主板过于复杂，如果需要关机，待重启到引导页时，手动按电源键关机，对系统和硬件都没有影响，黑苹果不完美，有时得适应。

10.13.6系统FCPX打开闪退意外退出的问题

主要是现在10.13.6版本的软件签名己经过期了，这个需要手动更新签名。

Nvidia Quadro P2200显卡无法安装黑苹果

对于这个问题，理论上来说，P2200是支持的，但安装过很多个，都是无法安装成功，所以老吴建议不要再用P2200了。

耗时一星期！华硕C621主板加至强 Gold 5218 CPU黑苹果安装成功，支持10.15.7

这就是最新的MACPRO配置啊,但价格远远比不上。

先来研究一下配置，Xeon至强Gold 5218是第一次安装，构架是Skylake-SP，由于是至强系列，统一当成HEDT配置来看待。主机是C621芯片组的，也是第一次遇到，完全没有头绪。内存无论多大，MACOS都能认，这个只要注意内存补丁就好，硬盘没问题，显卡免驱的，不用理会。

选定OC引导，也只有OC能搞掂这套配置，CLOVER估计内存都很难过，事实也证明了CLOVER卡在内存不会再往下走。

系统安装macos catalina 10.15.7，也可以安装macos 10.14.6，如果安装10.15.7，可以将机型改为macpsor 7.1提高性能。big sur没有试过，也由于远程的关系，不能作太多尝试，毕竟安装这一个都花了近一个星期。

安装时注意的事项

1，卡在 end randomseed

多数由内存引起，记得勾选上setupvirtualmap和rebuildapplememorymap还有protectuefiservices，不同的内存需要不同的组合，多尝试。

2，卡在8 acpi aml错误
多数是DSDT参数错误，认真修改RTC相关的DSDT补丁

3，卡在PCI CONFIGURATION END BRIDGES 8

加上npci=0x2000的参数

越简单越难搞！记联想ThinkPad X1 Carbon i5三代老笔记本安装黑苹果Mojave 10.14.6

发表于 2020-11-30 更新于 2021-12-27 分类于 笔记本 ， ThinkPad

客户一台联想ThinkPad X1 Carbon笔记本，应该是X1 CARBON 一代产品，CPU是i5三代的3317U,没有独显，核显HD 4000,Ivy Bridge架构。
主板联想 3448aw5 QS77的，4GB内存，1600X900分辨率显示屏，ALC 269声卡，没有自带有线网卡，无线网卡为6205无解。
180GB固态安装单系统，按这个配置来说，10.14.6 Mojave是最适合的。

下面讲讲安装时遇到的问题，根据配置，选用CLOVER引导，比较成熟，热补丁这些比较适合老机子，OC虽然新，但很多补丁需要手动，也就没有必要了。

HD 4000的核显非常好驱动，只需将ig-platform-id设置为0x01660004即可，其它都不用设置，因为没有独显，所以也不用制作屏蔽独显的SSDT。亮度调节丢SSDT-PNLF.aml进去即可。

ALC 269声卡个人建议用VOODOOHDA万能驱动，比较快捷，当然APPLEALC也可以设置ID为6，但APPLEALC驱动有时会掉，看个人选择。

因为没有有线网卡，无线网卡也无法驱动，所以建议客户买个黑苹果无线网卡，淘宝才10多块钱。

特别需要注意的是，内存修复补丁，如果出现重启后卡加号，建议升级成CLOVER R5127最新版本，用OpenRuntime.efi替代AptioMemoryFix.efi，这些内存补丁都太老旧了。

另外，不要用OC 6.3版，会出现OC引导界面卡死

安装好之后，核显驱动，亮度调节正常，声卡OK，触控板OK，电池电量有显示，基本完美。

EFI下载：https://efi.win2mac.top/share_1456.html

安装黑苹果准备工作之：更新BIOS

发表于 2020-11-26 分类于 教程

无论是台式机，还是笔记本，或者是工作站服务器主板，在安装黑苹果之前，首先要做的就是更新主板的BIOS到最新的版本。

这个尤其重要，涉及CPU的微码支持，各项硬件的兼容性，减少黑苹果安装中可能出现的问题，各种奇怪错误。

首先，可以用鲁大师的主板检测，查找主板现的BIOS版本，也可以进入BIOS之后查看主板信息。

根据主板，笔记本型号，到官网查询BIOS，下载后，有些可以直接运行更新，有些需要复制到U盘，进BIOS的更新工具升级。

黑苹果准备工作之一，就是更新BIOS，这是所有的基础，建议先做这一步。

微星Z370 KRAIT GAMING主板搭配I7-8700K N卡GTX 1060安装黑苹果注意事项

发表于 2020-11-26 更新于 2021-12-27 分类于 台式机 ， INTEL

配置图：

8代I7-8700K，微星Z370 KRAIT GAMING主板，32GB（两条16G 2400）,双NVME硬盘，一个970 EVO一个970 EVO PLUS，GTX 1060独显。

就以上的配置，安装黑苹果MACOS 10.13.6版，看配置是相当简单，但往往越简单越容易出问题，以下总结问题所在。

首先CPU没什么好说的，8代配置，核显默认在BIOS是关闭的，开不开启问题不大，反正10.13.6也不支持UHD 630.

然后就是主板微星Z370 KRAIT GAMING，这个要留意三点：第一，BIOS必须更新到最新版本，尤其重要，我安装的时候这块主板仍然是2017年的BIOS版本，各种出错，需要先到官网下载最新的2020版本升级更新。第二，关闭CFG LOCK，尤其重要，这块主板不能通过引导的补丁跳过CFG LOCK，需要在BIOS搜索CFG，将其禁用！第三，两块NVME硬盘，在硬盘模式中，SATA的热拔插禁用，SATA模式一定要是ACHI。

然后内存，别小看这两条32G内存，用CLOVER的话，往往就会卡++++号，或者

卡在End Randdomseed不动，又或者couldn’t allocate runtime area错误，这些都是不能通过内存的错误，与其一个个去试内存补丁，还不如直接换OC引导。

再者就是硬盘了，相当吓人的三星硬盘，对黑苹果极度不友好的970 EVO PLUS版，记得先升级硬盘固件到最新版本，怎么升级，百度一下就好了。

综合以上都解决之后，就可以一路畅通安装黑苹果，基本一步到位。

如何选择合适的黑苹果系统版本？

发表于 2020-11-24

很多人都喜欢更新升级最新版本的MACOS系统，如现在新出的BIG SUR 11，但就黑苹果来说，最新的未必是最适合自己电脑的，因为黑苹果不同于原装mac电脑，引导程序，补丁等都需要时间来适配和完善，如果单纯为了尝鲜，可以安装最新的，但实用性不大，如果用于工作，建议安装稳定成熟的版本。

macos版本对于黑苹果来说，现在最低的只能是macos 10.13.6也就是macos high sierra,低于这个版本的，除非有特殊需求，如FCPX 7的，否则都不建议再装低的版本了，很多软件都无法运行。10.13.6这个版本还有一个特别的是，NVIDIA显卡。如果你是N卡，如GTX 1080 1070 1060这些，目前只有10.13.6这个版本有驱动，其它都没有，所以也就是说，大多数N卡独显用户只能安装MACOS 10.13.6，并且无法升级更新，解决的方法只有一个，换AMD显卡。另外，10.13.6相对来说的确老了点，如果你是用于xcode和FCPX，只能安装较低的版本，应用商店是无法下载最新版本了，这个需要注意。

推荐cpu是3代的Ivy Bridge，内存小于4G的台式机或者笔记本，独显为n卡gtx 1080以下的台式机，安装macos 10.13.6。

接下来是macos mojave 10.14和catalina 10.15，这两个版本有很多相似性，目前来说，mojave我是不建议安装，建议直接上catalina，实测发现catalina流畅性比mojave还要好，而且版本高，兼容软件来说日后也比较好。如果你的CPU是Haswell 4代以上到现在的9代，内存在4G以上，安装catalina是最明智的，目前最新版本是catalina 10.15.7，直接食用。如果觉得4代和5代CPU运行catalina吃力，那可以安装mojave 10.14，自由选择。

另外就是big sur了，现在己经更新到正式版11。黑苹果来说，我现在并不建议更新，主要是引导还没有跟上，建议big sur更新到11.5这样，条件都成熟了才开始用。硬件配置至少10代CPU才是BIG SUR的标配。
