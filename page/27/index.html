戴尔 Inspiron 5280 i5-7Y54笔记本黑苹果引导OC 7.6 MONTEREY 12.1

发表于 2022-02-07 更新于 2022-08-10 分类于 笔记本 ， Dell_戴尔

配置

电脑型号 戴尔 Inspiron 5280 笔记本电脑
处理器 英特尔 Core i5-7Y54 @ 1.20GHz 双核
主板 戴尔 0CH4MP（7th Generation Intel Processor Family I/O - 9D4B 笔记本芯片组）
显卡 英特尔 HD Graphics 615 ( 128 MB / 戴尔 )
内存 8 GB ( 未知 LPDDR3 1867MHz )
主硬盘 镁光 _1100_MTFDDAV256TBN ( 256 GB / 固态硬盘 )
显示器 京东方 BOE075C ( 12.3 英寸 )
声卡 瑞昱 @ 英特尔 High Definition Audio 控制器
网卡 英特尔 Dual Band Wireless-AC 3165

说明

这是一个平板，只有两个type c口，没有usb口的。原配的键盘可以用，但键盘上的触控板用不了。触屏可以用。
核显，声卡，电池电量，变频都可以，基本正常使用。
有一个需要特别注意的，就是bios的安全引导这个非常麻烦，需要把oc引导的boot和oc文件夹放到system引导分区中，其中windows自带的boot文件夹请先备份一下，然后用oc的boot文件夹替换。再用esayuefi或者diskguies新建oc引导指向oc/opencore.efi。
Bios中，需要在secuity中的secure boot关掉安全引导，boot中还有一个secure boot也要关掉，引导顺序#1要改为新建的OC引导。多试几次，有时重启错误，或者windows引导，安全引导又会自动开启，就只能进win，这时要再设置一次，反正这个平板的引导好像只认windows引导，需要多留心。
另外，这个是带sim网卡的，目测是不能用的了，摄像头也不行，没有驱动。
