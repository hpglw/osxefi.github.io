i7-7700K - 华硕 PRIME B250-A - GTX 1060台式机黑苹果引导EFI OC7.3 High Sierra 10.13.6

发表于 2021-10-14 更新于 2022-08-10 分类于 台式机 ， INTEL
配置

处理器 英特尔 Core i7-7700K @ 4.20GHz 四核
主板 华硕 PRIME B250-A（B250 芯片组）
显卡 Nvidia GeForce GTX 1060 6GB ( 6 GB / 影驰 )
内存 32 GB ( 麦光MG DDR4 2400MHz )
主硬盘 三星 SSD 850 EVO 250GB ( 250 GB / 固态硬盘 )
显示器 戴尔 DELA0D8 DELL P2217H ( 21.5 英寸 )
声卡 瑞昱 ALC887 @ 英特尔 High Definition Audio 控制器
网卡 瑞昱 RTL8168/8111/8112 Gigabit Ethernet Controller / 华硕

特别提示：
BIOS:
1,CPU设置：cpu电源关闭cfg lock（默认一般是关闭的）
2，北桥——大于4g的地址解码空间：开启
3，北桥——显示设置——初始化igpu开启
4，启动——快速启动关闭，csm关闭

此配置开启了核显的辅助运算加速，对于剪辑视频等应用等有好处。需要注意的是，bios里一定要开启igpu，否则核显是默认被屏蔽的，在有独显的情况下。另外，该配置的独显可能不支持双显示器输出，也有可能是该型号的问题。

E3-1230 v2--微星 B75MA-E31--RX 560黑苹果引导台式机EFI OC7.3 BIG SUR 11.6

发表于 2021-10-14 更新于 2022-08-10 分类于 台式机 ， INTEL
配置

微星 B75MA-E31
CPU：E3-1230 v2
内存：16G
显卡：RX 560
显示器：1080P
硬盘:240GB
网卡：博通bcm943224pciebt2_4.0
声卡：ALC887

特别提示：
该efi包含博通bcm943224pciebt2_4.0免驱无线网卡驱动，如没有这个网卡，不影响，可用有线网卡，驱动一般也支持其它其它博通免驱网卡的。
Usb己定制，适用绝对大部分前置usb接口的机箱
注意，如果cpu不是E3-1230 v2，可能导致无法进入系统，但一般来说3代的e3都可以用，可能变频没这么好，因为这个efi的变频文件是为这个cpu定制的，而主板b75b85一般都支持不限品牌。
显卡为rx 560，免驱显卡都可以往里上，不需要动efi配置，己经加速显卡了。
