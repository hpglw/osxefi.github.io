i3-8100 - 技嘉 B360 M AORUS PRO-CF核显HDMI 4K黑苹果安装引导EFI OC7.4 BIG SUR 11.6

发表于 2021-12-15 更新于 2022-08-10 分类于 台式机 ， INTEL
配置

处理器 英特尔 Core i3-8100 @ 3.60GHz 四核
主板 技嘉 B360 M AORUS PRO-CF ( B360 芯片组 )
内存 12 GB ( 芝奇 DDR4 2666MHz / 金士顿 DDR4 2400MHz )
主硬盘 ADATA SP920SS ( 256 GB / 固态硬盘 )
主显卡 英特尔 Gaussian Mixture Model - 3E91 ( 128 MB / 技嘉 )
显示器 IPA2800 E281UA ( 28 英寸 )
声卡 瑞昱 ALC892 @ 英特尔 High Definition Audio 控制器
网卡 英特尔 Ethernet Connection I219-V / 技嘉

BIOS里，USB程序把XHCI HAND OFF开启

芯片组里把DVMT 64MB改为128MB（上4K要，不上4K,64MB够）DVMT TOTAL改为MAX

进MACOS之前要在OC引导页按空格，先CFG LOCK，之后按Y，关闭CFG LOCK，不走这一步会卡在开始处LOG不动。

i5-4690 - 微星b85-g41 pc mate - gtx 1060黑苹果efi oc7.4 macos 10.13.6

发表于 2021-12-15 更新于 2022-08-10 分类于 台式机 ， INTEL
配置

微星b85-g41 pc mate
CPU：i5-4690
内存：16G
显卡：HD4600+GTX 1060
显示器：1080P
硬盘:512GB
声卡：ALC887

Bios里记得要在oc/cpu特征/cfg lock关闭
如果有核显，在bios里打开多显示支持，显存改为64mb
如果是gtx 10xx独显接两个显示器，可能进系统黑屏，这时应单接一个，再接另一个，dp单接会没显示，可以试一下hdmi或者dvi，这是主板没有csm模块的原因。
Cfg lock关闭后变频完美，gtx 10xx独显不支持解压，需要硬压的要打开核显，引导配置里己经把核显的参数写进去了，支持所有4代cpu核显。

Razer Blade 15 i7-8750H笔记本黑苹果安装引导OC 7.4 BIG SUR 11.6

发表于 2021-12-15 更新于 2022-08-10 分类于 笔记本 ， Razer Blade_雷蛇
配置

Razer Blade 15 2019
CPU：i7-8750H

特别提示：
请注意，该机型的分辨率为1080P的，如果是2K或者4K屏，这个引导不适用
BIOS里把雷电选项的认证改为NO SECUITY，安全引导禁用
进系统之前，需要在OC引导页按空格，选CFG LOCK，之后按Y，关闭CFG LOCK，然后再进MACOS，如果不关，就卡在LOG上不动。
雷电口接普通TYPE C设备是可以直接识别的，但要外接C转HDMI显示器，需要接着显示器，然后重启之后才能显示。
本身内置的HDMI接口无法外接显示器，因为是直连了独显。
INTEL的网卡WIFI可以用，蓝牙不怎么行，有时能识别有时不行，USB端口经过定制了，所有口都能识别。
电池，触控板，多指是好的，基本比较完美。

ASUS FX505GT笔记本 I7-9750H黑苹果安装引导EFI OC7.4 MONTEREY 12.0.1

发表于 2021-12-15 更新于 2022-08-10 分类于 笔记本 ， ASUS_华硕
配置

ASUS FX505GT笔记本
I7-9750H
镁光S2200（屏蔽）+金士顿固态（安装黑苹果）
UHD630+GTX 1650
ALC 235
INTEL WIRELESS-AC（无线可以，蓝牙不行）
显示器：1080P（120HZ）

特别提示：
BIOS里要把DVMT改为64MB（一般默认就是），安全启动关掉。

该配置中的NVME是镁光S2200，引导中是屏蔽的，如果你的NVME不是这个，也不是PM981，直接把OC/ACPI里的SSDT-DPM981.aml这个文件去掉就行

惠普暗影精灵2 PRO（OMEN by HP Laptop）笔记本i7-6700HQ黑苹果引导OC7.4 BIG SUR 11.6独显驱动

发表于 2021-12-15 更新于 2022-08-10 分类于 笔记本 ， HP_惠普
配置

处理器 英特尔 Core i7-6700HQ @ 2.60GHz 四核
主板 惠普 8294 ( 100 Series/C230 Series 芯片组 Family - A14E )
内存 16 GB ( 海力士 DDR4 2400MHz / 金士顿 DDR4 2400MHz )
主硬盘 闪迪 SD8SNAT-128G-1006 ( 128 GB / 固态硬盘 )
显卡 AMD Radeon RX 460 ( 4 GB / 惠普 )
显示器 LG LGD0532 ( 15.5 英寸 )
声卡 瑞昱 @ 英特尔 High Definition Audio 控制器
网卡 英特尔 Dual Band Wireless-AC 7265

BIOS里关掉安全启动

这个HP的BIOS不能自动识别OC引导，并只能认WINDOWS的引导也就是WINDOS BOOT MANAGER，要用DISKGUEIS或者EASYUEFI新建一条OC的引导，然后进去BIOS系统设置，操作系统的启动管理员，把OC的引导按F6移到最上面，记得再按F10保存住才行，
DISKGUEIS或者EASYUEFI建立的OC引导指向OPENCORE.EFI。

最神奇的地方在于这台笔记本有RX460的独显，是可以完全驱动的！！

读卡器可用，WIFI可以用，记得用HELIPORT工具，蓝牙可能不行，核显，声卡（从WIDNOWS重启进MACOS会没声，关机后开机直接进MACOS才有声音），触控板，电池，USB己经定制过了，摄像头可用，变频正常，睿频OK。
