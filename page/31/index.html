为什么PM981，PM981A不能支持安装黑苹果？

发表于 2021-01-19 分类于 教程

咨询老吴安装黑苹果的配置中，10个至少有2个是使用三星PM981 PM981A这块NVME固态的。老吴也一直强调这个型号不支持安装黑苹果，虽然说有人说己经成功安装（利用恢复版然后打补丁），但老吴绝对是不推荐的，为什么？

  • 第一，包括OC的开发者，都明说了这款固态的固件存在黑苹果系统兼容问题，连他们都不推荐，除此之外还有镁光的S2200.
  • 第二，虽然新版的NVMeFix.kext对pm981有修复作用，但依然存在无法进系统，或者进了系统后不久便卡死，自动重启的问题，这导致了不可预料的问题太多。
  • 第三，网上有的打补丁方法，通过恢复版系统安装后添加ssdt-nvme.aml和HackrNVMeFamily-10_12_6_sec2018-001.kext补丁的方法，其原理跟nvmefix.kext基本一样，虽然说可以进系统，但也存在这个那个的问题，而这些问题在使用中便会浮现出来，严重影响使用。
  • 第四，假设通过以上方法成功进入系统，对于PM981来说，只要利用磁盘分区工具进行分区，无论使用哪种补丁，都会直接卡死，这就表明了这款固态对于黑苹果来说是有问题的。
  • 第五，哪怕装在苹果MAC电脑中，也存在问题，何况黑苹果。

综合以上的问题，老吴不建议有上面两个型号的固态的用户安装黑苹果，别再折腾网上所谓的方法，最后装完还使用中一堆问题。有PM981的用户，一是可以换其它品牌，二是可以作为WINDOWS的盘使用，但进了MACOS就要屏蔽掉。

挖矿潮A卡价格飙涨，黑苹果独显还有什么好选择？

发表于 2021-01-19 分类于 教程

2021年，比特币飙涨，ETH也跟着涨到最高点。显卡用来挖矿，尤其是AMD的，价格也随之水涨船高，以至于几年前的RX 460 470 480 560 570 580 590这些老卡都被疯抢。这些显卡装黑苹果是免驱的，所以我一般都推荐使用。以前矿难的时候，一张RX 480也就几百元，有时甚至300元可以买到，现在也涨到1300 1400一张。

装黑苹果推荐用A卡，N卡因为诸多因素并不推荐，很难找到一张真正免驱的显卡，而AMD很多都可以直接免驱，上最新的CATALINA或者BIG SUR系统，甚至于，以后的系统都可能直上免驱。

那么，还有选择吗？答案还是有的。那就是挖矿的人都看不上，用不上的老显卡，如HD 7000 系列，和HD 8000系列，这些独显二手价格都还算实惠，100到300之间，可以接受的范围，最重要的是，是可以免驱的，对于黑苹果来说，性能也算不错，对于视频加速，剪辑，配合核显，有相当不错的表现。

个人推荐HD 7850，以上这些都可直上BIG SUR系统的。另外，R7/R9系列也可以免驱，但不确定因素太多，有些型号需要仿冒ID，出错的机率太高了，所以不作推荐。

在挖矿导致显卡价格升高的情况下，以上型号的显卡不实为好的选择，但全部都是二手卡，能否选到一张心意的，全看人品了。

Big Sur正式版11.1已经不再支持删除系统快照获得系统盘读写权限了

发表于 2021-01-15 分类于 教程

老吴经过多次尝试，网上的方法都用过了。目前BIG SUR 11.1正式版本可以确认不能用现在的方法删除系统快照从而获得系统盘读写权限。

在MACOS恢复模式中挂载系统盘，然后删除快照会提示insufficient privileges权限不足，原因是快照名上面己经清楚写明purgeable:no，也就是不可删除！

假设己经无法再这个方法挂载系统盘，从而修改文件，那我们要做的，就反向加大安全性了，其实没什么需要往系统盘里加的，除了HIDPI，但目前一键HIDPI的作者己经更新了脚本，无须往系统盘里写东西，在这里：
https://github.com/xzhih/one-key-hidpi

玩黑苹果的，可能还是期待后续会有大神更新修改方法。但目前来说，还没有。
