如何在Macos下制作winpe启动U盘

发表于 2020-01-13 更新于 2021-12-27 分类于 教程

试过很多方法在macos下制作一个winpe启动U盘，都不成功，后来研究出这么一个方法是可用的，这对于黑苹果用户非常有用，有很多黑苹果用户安装的都单系统，如果引导出问题，就有可能需要一个winpe的急救U盘。在这里分享给大家，我的方法是在windows下制作一个微PE的U盘，然后在macos下打包成dmg镜像，再用Etcher这个工具进行还原。

首先，需要准备一个8G以上的U盘。
然后下载镜像：

wepe for mac.dmg

链接:https://pan.baidu.com/s/1rahxi1SH9o1BulZGqJmsUg 密码:lhvu
完全基本微PE系统（官网：微PE工具箱 - 下载）

balenaEtcher：

百度云链接:https://pan.baidu.com/s/1WBdhcpc-PHJ2k-4Lnm2zYA 密码:5mbj
或者：Etcher for Mac-Etcher Mac版下载 V1.5.0-PC6苹果网

下载之后，打开balenaEtcher.dmg

拖到应用程序文件，再打开balenaEtcher

选择wepe for mac.dmg镜像，然后把U盘插进去，选择目标U盘（注意，会清空U盘），然后再点Flash!，等待片刻即可刻录完成。
这样一个微PE启动U盘，在macos下就做成了，开机选UEFI U盘启动，就可以进到微pe环境，进行我们的操作和修复了。

N卡有哪些可以安装黑苹果的？gtx1660黑苹果驱动有吗？

发表于 2020-01-10 分类于 教程

首先，可以明确地说，GTX 1660不支持黑苹果，无论哪个系统版本，都没有驱动支持！

虽然，使用GTX 1660可以引导进入系统，但因为显卡没有驱动，往往显存只有7M，加速也没有，只能当个文本处理电脑使用，但xcode这些敲代码软件是没有问题的。视频处理，设计这些就不要想了。

N卡目前己明确表示不再与苹果公司合作，也不会再发布基于macos系统的驱动程序，所以以后都不要指望有驱动更新了。

可以驱动的最高的N卡是1080 TI，也就是1000系的GTX系列显卡，如1050 1060 1070 1030这些，而且只有MACOS 10.13.6最高这些版本有驱动，要安装10.14.6 MOjave版本也是驱动不了的。

所以RTX，16XX 2XXX的N卡用户，黑苹果基本是无望的了，除非你只是使用苹果系统做代码开发的，那是可以正常进系统的，只是显卡没有驱动，分辨率也是一个问题。

对黑苹果的一些误解，安装黑苹果系统会烧毁芯片吗？

发表于 2020-01-10 分类于 教程

经常会有人问老吴，如果安装黑苹果系统之后，会不会烧芯片，CPU，主板什么的，对硬件有什么损害？
我只想说，无稽之谈！
就算是再厉害的计算机病毒，也只是损坏系统软件或者操作系统。系统坏了，重装一下就好，硬件是不会损坏，只有老化和本身的元器件故障才会损坏硬件，但这些都是外力不可抗因素造成的，与软件并没有关系。

双系统不想要黑苹果了，如何删除黑苹果系统教程

发表于 2020-01-09 更新于 2021-12-27 分类于 教程

很多人用不习惯macos系统，想要删除黑苹果系统，操作方法如下，在双系统环境中，进入windows，有三个地方要操作的，删除黑苹果就很干净了。
需要准备两个工具：
DiskGenius
EasyUEFI
1，首先，打开计算机的管理——磁盘管理


把安装了macos黑苹果的分区整个删掉！然后可以合并，也可以新建成ntfs格式，这个随意。

2，打开DiskGenius工具，找到ESP盘，里面有个EFI文件夹，把EFI文件夹中的CLOVER文件夹删掉。

3，安装EasyUEFI工具，打开找到 CLOVER的引导项，删掉！

就这样，黑苹果就彻度删除了！

再谈谈三星PM981和EVO 970 PLUS安装黑苹果的问题

发表于 2020-01-09 更新于 2021-12-27 分类于 教程

虽然说这两款NVME固态的确性价比不错，很多笔记本，如小米，戴尔，联想都选用了这款硬盘。

PM981型号开头为MZVLB，只要有MZLVB就是PM981，还有PM981A两种。但就安装黑苹果来说，目前为止不可以，PM981和PM981A绝对是不行的。国内有些论坛说己经解决，那是不可信的，因为老吴己经安装太多了，无非表现都是开机能进系统，但会莫机卡死，自动重启等一系列奇怪的问题，反正就是不能正常使用，也用过各种说可以解决的方法，无果！从国外的大神反应来看，都是保持沉默，要么建议更换更换其它硬盘

如果你的笔记本中有PM981，也不是说一定安装不了，一般现在的笔记本都可以安装两个硬盘，只要将黑苹果安装在另一块上面，然后进黑苹果之后屏蔽PM981即可，屏蔽的方法很多，有DSDT法，也有开机不挂载法，老吴认为开机不挂载法是最方便的，也是最简单的
教程可以看：https://win2mac.top/diable-pm981-on-hackintosh.html

另外还有一个三星EVO 970 PLUS，目前这一块可以通过刷新最新的固件，安装黑苹果是没有问题，教程在这里：https://win2mac.top/970-EVO-Plus-firmware-flash.html
但是，目前发现了在macos catalina 10.15.2版本中，这块硬盘又出问题了！！再次不兼容！具体表现就是无法进系统，卡在苹果LOGO上，还没有解决方法，只能用10.15.2以下的macos版本。

老吴建议，三星这两款固态的，都不要安装黑苹果，就算安装，也只能在其它硬盘上装，然后还是屏蔽掉，也许只能等固件更新，毕竟三星也不会为了黑苹果而去修改固件，更多的还是为windows系统考虑。

黑苹果怎么屏蔽三星PM981

发表于 2020-01-09 更新于 2021-12-27 分类于 教程

首先，进系统后，第一时间，去启动台——其它

打开磁盘工具，把三星PM981的所有分区都卸载掉先，否则可能会卡死或者自动重启，无法进行后续的屏蔽工作。

1，打开终端，输入diskutil list
2，找到PM981的硬盘信息：

每个人的分区信息，都不一样，我们只需要找到可见的分区，如C盘D盘这些，在INDENTIFIER参数中，如C盘的分区是disk1s1，记下来。
3，diskutil info disk1s1，然后找到Volume UUID值，复制出来
4，再打开终端，输入sudo vifs
5，在vim编辑器中按i，然后在最后一行输入UUID=Volume UUID none ntfs noauto，有几个分区，就有几行。
6，保存vim退出，重启之后，开机自动卸载PM981所有磁盘分区，屏蔽成功。
