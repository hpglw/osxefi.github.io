Nvidia显卡黑苹果驱动大全，附下载地址

发表于 2019-11-15 更新于 2022-07-01 分类于 驱动

Nvidia显卡在苹果系统macOS的驱动叫web-driver，除了一些免驱的N卡，如GTX 760，其余的大部分需要安装这个驱动才能正常，否则卡得一逼。
目前苹果己经不再使用Nvidia显卡了，转跟AMD合作，较新的机型统一采用AMD的显卡，这就导致了驱动只停留在10.13.6版本，Nvidia公司估计觉得再开发也没有意义了。
Nvidia web driver目前支持最高家用显卡是1080 TI，专业类显卡还没怎么研究过，但大部分都支持。像GTX RTX 16xx 2xxx这些，不用想了，无缘黑苹果。

首先，安装驱动前需要确认自己系统的版本，最低为OS X 10.8.3 Mountain Lion，最高为macOS 10.13.6 High Sierra。也许，可能，以后会支持Mojave，也许可能，永远都不会支持。

打开关于本机

在版本号这里点一下

就会出现最终的版本尾号

如，老吴的是macos 10.13.6(17G8037)
